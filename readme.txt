To run this program,

Please install "Node.js" and "Mongodb",first.

Open two command prompts,
  1. run "node server" at the correct path that server.js located.
  2. run "mongod" at the correct path that mongod.exe located.
     (ex. "C:\Program Files\MongoDB\Server\3.2\bin")

Open a web browser, and type "localhost:3000".
