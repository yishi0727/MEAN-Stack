This project is developed with the objective to keep up-to-date in real time the most
effective contact information for the fast pace communication over the internet.
This is aimed to a financial management firm and it also has wide applications in other
industries, like telecomm, manufacturing, retail trade and where the customer contact
details would be easily saved and updated.

To run this program,

Please install "Node.js" and "Mongodb",first.

Open two command prompts,
  1. run "node server" at the correct path that server.js located.
  2. run "mongod" at the correct path that mongod.exe located.
     (ex. "C:\Program Files\MongoDB\Server\3.2\bin")

Open a web browser, and type "localhost:3000".
